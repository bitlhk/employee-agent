### 步骤1: 创建缺失的高优先级模板（优先）

#### 1.1 创建"PDF层级字段保险责任.md"
```markdown
# 保险责任提取模板

《合同》的层级信息如下：
{合同层级部分输入}

从《合同》的层级信息中提取保险责任相关信息，包括保险责任名称、保险金额、交费标准、约定项、约定值等字段。

一、字段提取规则：

"保险责任": 保险责任的名称，如"意外身故保障"、"意外残疾保障"、"门(急)诊医疗保险金"等
"保险金额": 保险金额的数值，如"500000.00"、"20000.00"等
"交费标准": 交费标准的数值，如"200.00"、"35.00"等
"约定项": 约定项的名称，如"等待期"、"门(急)诊次免赔额"、"赔付比例"等
"约定值": 约定项对应的值，如"30天"、"0元"、"100%"等

二、表格识别规则：

1. 找到包含"保险责任"、"保险金额"、"交费标准"、"约定项"、"约定值"字段的表格
2. 识别表格的层级结构（险种名称 > 保险责任 > 约定项）
3. 处理合并单元格（多个约定项对应同一个保险责任）
4. 提取所有险种的保险责任信息

三、输出格式：

result = {
    "保险责任列表": [
        {
            "险种名称": "团体意外伤害保险",
            "保险责任": [
                {
                    "保险责任名称": "意外身故保障",
                    "保险金额": "500000.00",
                    "交费标准": "200.00",
                    "约定项": []
                },
                {
                    "保险责任名称": "意外残疾保障",
                    "保险金额": "500000.00",
                    "交费标准": "",
                    "约定项": [
                        {"约定项名称": "伤残标准", "约定值": "人身保险伤残评定标准"}
                    ]
                }
            ]
        },
        {
            "险种名称": "附加团体意外伤害医疗保险（B款）",
            "保险责任": [
                {
                    "保险责任名称": "门(急)诊医疗保险金或住院医疗保险金",
                    "保险金额": "50000.00",
                    "交费标准": "35.00",
                    "约定项": [
                        {"约定项名称": "门(急)诊次免赔额", "约定值": "0元"},
                        {"约定项名称": "门(急)诊赔付比例", "约定值": "100%"},
                        {"约定项名称": "住院次免赔额", "约定值": "0元"},
                        {"约定项名称": "住院赔付比例", "约定值": "100%"},
                        {"约定项名称": "是否使用公共保额", "约定值": "是"}

                }
            ]
        }
    ]
}
```
