# 多模态文本坐标定位模板

请从图片<image>中识别文本并定位其坐标位置，要求：

---

## 一、坐标定位规则

### 1.1 文本识别
- ✅ 识别图片中的所有文本内容
- ✅ 保持原文的格式和顺序
- ✅ 识别表格中的文本内容
- ✅ 保留特殊标记和符号

### 1.2 坐标定位
- ✅ 定位每个文本块在图片中的坐标位置
- ✅ 坐标格式：左上角(x, y)，宽(width)，高(height)
- ✅ 坐标单位：像素(pixel)
- ✅ 标注页码信息

### 1.3 文本块定义
一个文本块包含：
- **文本内容**: 文本块中的文字内容
- **坐标位置**: 文本块在图片中的矩形区域
- **页码**: 文本所在的页码

---

## 二、重点定位文本

### 2.1 保险责任相关文本
重点定位以下文本的坐标：
- **险种名称**: 如"团体意外伤害保险"、"附加团体意外伤害医疗保险（B款）"
- **保险责任**: 如"意外身故保障"、"意外残疾保障"、"门(急)诊医疗保险金或住院医疗保险金"
- **保险金额**: 如"500000.00"、"20000.00"
- **交费标准**: 如"200.00"、"35.00"
- **约定项**: 如"等待期"、"门(急)诊次免赔额"、"赔付比例"
- **约定值**: 如"30天"、"0元"、"100%"

### 2.2 层级信息相关文本
重点定位以下文本的坐标：
- **层级名称**: 如"上海及青岛员工有医保"、"外籍"、"临时工"
- **被保险人属性**: 如"主被保险人"、"连带被保险人"
- **层级保费**: 如"2750.00"、"745.00"、"216.00"
- **方案序号**: 如"01"、"05"、"06"

### 2.3 特约相关文本
重点定位以下文本的坐标：
- **特约标题**: 如"特别约定"、"方案特约"、"个人特约"
- **特约内容**: 特约描述的具体内容

---

## 三、坐标格式

### 3.1 矩形坐标
每个文本块的坐标用一个矩形表示：
```python
{
    "x": 左上角X坐标,
    "y": 左上角Y坐标,
    "width": 矩形宽度,
    "height": 矩形高度
}
```

### 3.2 坐标示例
```python
{
    "text": "意外身故保障",
    "coordinates": [
        {"x": 100, "y": 50, "width": 200, "height": 30}
    ],
    "page_number": 11
}
```
