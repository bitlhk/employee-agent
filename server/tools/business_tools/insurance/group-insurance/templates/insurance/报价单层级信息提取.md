# 报价单层级信息提取模板

《报价单》内容如下：
{报价单输入}

从《报价单》中提取层级和方案的对应关系、方案名称、方案保费等信息。

---

## 一、字段提取规则

### 核心字段
- **"层级名称"**: 层级的名称，如"员工"、"子女"、"配偶"、"外籍"、"临时工"等
- **"方案名称"**: 方案的名称，如"方案A"、"方案B"、"上海及青岛员工有医保"、"外籍"等
- **"方案序号"**: 方案的序号，如"01"、"02"、"05"、"06"等
- **"方案保费"**: 方案的保费，如"2750.00"、"745.00"、"216.00"等
- **"被保险人属性"**: 被保险人的属性，如"主被保险人"、"连带被保险人"

---

## 二、报价单结构识别

### 2.1 层级信息位置
报价单中的层级信息通常位于：
- "层级对应方案"部分
- "保障方案"部分
- "方案明细"部分

### 2.2 方案信息位置
报价单中的方案信息通常位于：
- "方案名称"行
- "方案保费"行
- "保险责任表"标题行

### 2.3 特约信息位置
报价单中的特约信息通常位于：
- "特约"部分
- "特别约定"部分
- "方案特约"部分

---

## 三、输出格式

严格按照以下Python字典格式输出：

```python
result = {
    "层级与方案关系": [
        {"层级名称A": ["方案名称1", "方案名称2"]},
        {"层级名称B": ["方案名称1"]}
    ],
    "层级名称顺序": [
        "层级名称A", "层级名称B"
    ],
    "方案名称": [
        {"名称": "方案名称A", "序号": "01", "保费": "2750.00"},
        {"名称": "方案名称B", "序号": "02", "保费": "3020.00"}
    ],
    "方案特约": {
        "方案名称A": "方案特约内容...",
        "方案名称B": "方案特约内容..."
    },
    "报价单特约": "报价单整体特约内容..."
}
```

---

## 四、提取要点

### 4.1 必须完整提取
- ✅ 所有层级名称和对应的方案名称
- ✅ 所有方案名称、序号、保费
- ✅ 所有方案特约内容
- ✅ 报价单整体特约内容

### 4.2 禁止行为
- ❌ 禁止修改、归纳、删减原文内容
- ❌ 禁止遗漏任何层级或方案
- ❌ 禁止将方案特约内容省略或合并

### 4.3 特殊处理
- **层级与方案关系**: 每个层级对应哪些方案，以列表形式输出
- **方案特约**: 每个方案的特约内容，以字典形式输出（方案名称 -> 特约内容）
- **报价单特约**: 报价单整体的特约内容（如既往症、职业类别等）
